AthosHun\HTMLFilter\Configuration

  • Tags can be allowed or disallowed
  • Attributes can be restricted with regular expression
  • Allowed attribute implies allowed tag
  • Allowing an already allowed tag keeps attribute restrictions
  • Multiple tags can be allowed
  • Multiple attributes can be allowed

AthosHun\HTMLFilter\HTMLFilter

  • Plain text without any html remains unchanged
  • Disallowed tags are removed
  • Allowed tags are kept
  • Disallowed attributes are removed
  • Allowed attributes are kept
  • Allowed attributes not matching a regexp are removed
  • H t m l entities are left unchanged
  • Nodes are copied recursively
  • Html comments are preserved
  • Invalid markup is ignored
